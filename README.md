# Coconut Forest - Svelte Website

Website modern untuk perusahaan exportir kelapa, dibangun dengan **SvelteKit** dan **Bun**.

## Quick Start

```bash
# Install dependencies
bun install

# Run development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```

## Setup Images

Copy semua gambar ke folder static:

```bash
cp -r images/* static/images/
```

## Documentation

Untuk panduan lengkap dan guidelines untuk developers/agents, lihat **[AGENTS.md](./AGENTS.md)**.

## Tech Stack

- **SvelteKit** - Full-stack framework
- **Bun** - Package manager & runtime
- **Vite** - Build tool
- **Vanilla CSS** - Styling dengan CSS variables

---

**Note**: Project ini menggunakan **Bun** sebagai package manager. Jangan gunakan npm/yarn/pnpm.
